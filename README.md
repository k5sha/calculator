## ⚙️ The lab
**Coding Theory Lab** is a modern, high-performance web application designed for educational and professional analysis of error-correcting codes. Built with **Astro 5**, it provides interactive calculators for complex algorithms like Recurrent codes, Hamming codes, and more.


<img width="1200" height="630" alt="og" src="https://github.com/user-attachments/assets/4ce4ba32-793f-4cb5-a673-2234a16a8757" />

[**Live Demo**](https://calculator.k5sha.xyz/)

## ✨ Features

* **Interactive Calculators**: Real-time generation and decoding of error-correcting codes.
* **Blazing Fast**: Optimized with Astro for near-instant load times and minimal JavaScript overhead.
* **Global Search**: Quickly find algorithms using the `/` hotkey.
* **Modern UI**: A clean, "glassmorphism" inspired interface using **Plus Jakarta Sans** and a sleek blue-themed design.
* **Responsive**: Fully functional on mobile, tablet, and desktop devices.
* **SEO Optimized**: Structured data (JSON-LD) for better visibility in search engines.


## 🛠 Tech Stack

* **Framework**: [Astro 5](https://astro.build/) (Static Site Generation)
* **Language**: [TypeScript](https://www.typescriptlang.org/)
* **UI Components**: Native Web Components & [Preact](https://preactjs.com/)
* **Styling**: Modern CSS with CSS Variables and Flex/Grid layouts
* **Icons**: Custom SVGs & Lucide Icons
* **Font**: [Plus Jakarta Sans](https://fonts.google.com/specimen/Plus+Jakarta+Sans)

## 🚀 Getting Started

### Prerequisites

* **Node.js** (v18.17.1 or higher)
* **npm** or **pnpm**

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/k5sha/coding-theory-lab.git
    cd coding-theory-lab
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```
